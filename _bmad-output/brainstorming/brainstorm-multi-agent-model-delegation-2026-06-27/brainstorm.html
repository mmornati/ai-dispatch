Invented Techniques 2 brand-new

INVENTED

Agent Mirror Protocol

Every agent has a mirror/reflection agent that reviews its output before passing it downstream. Not a human review — an automated adversarial check. The primary agent produces output; its Mirror checks for quality, coverage, edge cases, and style violations. Only on pass does the output go to the next agent. Quality gates without human bottlenecks, and each Mirror can use a different (cheaper) model.

INVENTED

Inversion of Control Flow

Start from the final desired outcome (e.g. "deployed PR with passing CI") and work backwards in reverse: what agents, in what order, with what inputs, produce that outcome? Map the dependency chain from output to input. Finds gaps the forward approach misses — discovers missing dependencies, implicit ordering requirements, and agents that should be involved but weren't planned.
